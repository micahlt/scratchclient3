Scratch Client 3


version 0.8.0

Scratch Client 3 is a reimagined version of the Scratch webpage. It’s based on pure HTML, CSS, and Javascript as well as the Scratch API. It’s hosted on GitHub and runs using GitHub pages. This is an unofficial project run by @-Archon-, @-Ekmand- and @smileycreations15.





Scratch Client 3 is not related to nor affiliated with Scratch or the Scratch Team.
